Project Overview

This project presents an interactive Sales Performance Dashboard built using Power BI to analyze sales trends, profitability, product performance, and regional contributions. The dashboard enables business users to monitor key metrics and identify actionable insights for data-driven decision-making.

Features
✔️ KPI Cards displaying Total Sales, Profit, Orders, and Quantity Sold
✔️ Monthly Sales Trend Analysis
✔️ Sales Distribution by Region
✔️ Category-wise Sales Comparison
✔️ Top 10 Best-Selling Products
✔️ Profitability Analysis using Scatter Plot
✔️ Interactive Filters for Region, Year, and Category
✔️ Dynamic insights section highlighting major business findings
Tools & Technologies
Power BI
Power Query
DAX
Key Insights
Electronics generated the highest overall sales.
The East region contributed significantly to revenue growth.
Monthly sales showed fluctuations with peak performance during mid-year and year-end periods.
A few products accounted for a major share of total sales, indicating opportunities for targeted marketing strategies.
